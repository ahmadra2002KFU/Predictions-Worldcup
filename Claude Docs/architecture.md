# نظرة معمارية — مفيد لتوقعات كأس العالم

تطبيق ويب عربيّ أولاً لفاعلية توقّع مباريات كأس العالم بين مجموعة أصدقاء/زملاء.

## المكدّس التقني
- **Next.js 16** (App Router، Node runtime) + **React 19**.
- **PostgreSQL 17 + Prisma 7** (عبر driver adapter `@prisma/adapter-pg` — إلزامي في Prisma 7).
- **Tailwind CSS v4** + خط **Cairo** للعربية.
- **iron-session v8** لهوية بدون كلمة مرور (كوكي مُوقّع + صف Session في القاعدة).
- **motion** (Framer Motion) للحركات، **flag-icons** لأعلام الدول.
- **SSE** (Server-Sent Events) للتحديث اللحظي، **zod** للتحقق.
- **Docker + Caddy** للنشر.

## القرارات المعمارية الرئيسية
- **الهوية بدون كلمة مرور**: التسجيل بالاسم فقط (والبريد اختياري للاسترجاع). الكوكي يحمل معرّف جلسة عشوائياً، والاسم ليس هو الهوية (منعاً للانتحال). كل تسجيل/توقّع يُسجَّل في `AuditLog` مع IP والمتصفح والوقت لكشف الغش.
- **قفل التوقّع**: دالة واحدة `isLocked()` (`src/lib/matchLock.ts`) تُستخدم في الواجهة والخادم. الخادم هو مصدر الحقيقة: يعيد فحص القفل مقابل ساعة الخادم قبل الكتابة مباشرةً ويرفض بـ 409.
- **محرّك النقاط** (`src/lib/scoring.ts`): دالة نقيّة مغطّاة بـ 21 اختبار وحدة، تشمل قاعدة فرق الأهداف وحالة ركلات الترجيح في الأدوار الإقصائية.
- **التحديث اللحظي**: ناقل `EventEmitter` في الذاكرة + مسار `/api/sse` يبثّ للمتصفح؛ اتصال واحد لكل تبويب موزّع حسب القناة (chat / leaderboard).

## نموذج البيانات (prisma/schema.prisma)
`Participant`, `Session`, `Team`, `Player`, `Match` (يشمل حقول ركلات الترجيح)، `Prediction` (يخزّن النقاط)، `ChatMessage`, `AuditLog`.

## قواعد احتساب النقاط (٥ نقاط كحد أقصى للمباراة)
- توقّع النتيجة تماماً: **٣**.
- تعادل صحيح بنتيجة مختلفة: **٢**.
- فوز صحيح مع فرق أهداف مختلف بهدف واحد بالضبط: **٢**؛ غير ذلك مع اتجاه صحيح: **١**.
- اتجاه خاطئ: **٠**.
- أدوار إقصائية بركلات ترجيح: من توقّع الفريق المتأهّل **٢**، من توقّع تعادلاً **١**، من توقّع الخاسر **٠** (يُحسب على نتيجة ١٢٠ دقيقة لا الركلات).
- توقّع أفضل لاعب: **+١** (مستقل). توقّع صاحب أول هدف: **+١** (مستقل).

## بنية المشروع (مختصر)
- `src/app/*` — الصفحات ومسارات الـ API (عامة + `admin/(protected)` خلف بوابة كلمة مرور).
- `src/lib/*` — المنطق المشترك (db, session, adminAuth, matchLock, scoring, realtime, leaderboard, validation, audit, flags, format, labels).
- `src/components/*` — مكوّنات الواجهة (MatchCard, PredictionForm, ChatPanel, LeaderboardTable, TeamFlag، ومكوّنات admin/).
- `prisma/*` — المخطّط والترحيلات وبيانات dev التجريبية.
